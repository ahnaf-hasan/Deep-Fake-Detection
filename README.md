# Deepfake Detection Video Dataset

This dataset is designed for deepfake detection tasks, providing a comprehensive collection of video sequences for training and evaluating deep learning models that identify manipulated media. It is sourced from the official FaceForensics server, known for high-quality face manipulation datasets.

## Motivation

Most existing datasets focus on image-based manipulations, creating a gap in video-based resources. This dataset addresses that need by offering a large collection of both original and manipulated videos, enabling better research and model performance in real-world scenarios.

## Features

* High-quality video data
* Includes both real and manipulated content
* Suitable for deep learning applications
* Ready for training and evaluation workflows

## Use Cases

* Train deepfake detection models
* Evaluate existing algorithms
* Research video-based manipulation detection
* Develop new detection techniques

## Source

FaceForensics (official dataset)
